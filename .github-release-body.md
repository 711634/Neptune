# Neptune v1.0.0-beta

**Local Autonomous Agent Platform for macOS**

Neptune is a lightweight, battery-efficient autonomous agent platform that brings real multi-agent orchestration to your Mac. Uses Claude Code CLI (already authenticated) for execution—no external APIs, no billing.

## 🎉 What's New

✅ Multi-agent orchestration with task dependencies
✅ Dock pets tied to real agent activity (not mock data)
✅ Provider adapter system (Claude Code CLI, Desktop, VS Code)
✅ Battery-efficient design with Low Power Mode
✅ Skills + auto-detection for different project types
✅ Local-first architecture (~/.neptune/, zero external APIs)
✅ Comprehensive documentation and Windows roadmap

## 📥 Installation

1. Download **Neptune.dmg** (below)
2. Mount the disk image, drag Neptune.app to /Applications
3. Launch Neptune
4. Configure Claude Code CLI path (usually `/opt/homebrew/bin/claude`)

## 📋 Requirements

- macOS 13.0 (Ventura) or later
- Claude Code CLI installed and authenticated
- Active Claude subscription
